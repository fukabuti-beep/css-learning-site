ボックスモデル

CSSのすべての要素はボックスとして扱われます。content・padding・border・marginの4層構造と、box-sizingの違いを理解しましょう。

ボックスモデルとは

CSSのすべての要素はボックス（箱）として扱われます。このボックスは4つの層で構成されており、これをボックスモデルと呼びます。

🎯 ボックスモデルの構造

margin
border
padding
content
層説明プロパティ
contentテキストや画像などの実際のコンテンツwidth, height
paddingコンテンツとボーダーの間の内側の余白padding
borderパディングの外側を囲む境界線border
marginボーダーの外側の他要素との余白margin

🔍 ボックスモデルのコード - 1行ずつ読む

.card { width: 300px; padding: 20px; border: 2px solid #333; margin: 16px auto; }
width: 300px → コンテンツ（中身）の幅を300ピクセルにする
padding: 20px → コンテンツの内側に20pxの余白を作る（上下左右すべて）
border: 2px solid #333 → 2ピクセルの実線ボーダー（太さ スタイル 色の順）
margin: 16px auto → 上下16px、左右autoで水平中央揃え
  • padding vs margin の違い：paddingは内側（背景色が適用される）、marginは外側（背景色なし、他要素との距離）
  • margin: 0 auto：上下0、左右「自動」でブロック要素を中央揃えにする定番コード

box-sizing: border-box

デフォルトではbox-sizing: content-boxなので、widthにpaddingとborderが含まれません。border-boxを使うと直感的にサイズ管理できます。

CSS
/* 全要素にborder-boxを適用（モダンCSSの定番）*/
*, *::before, *::after {
  box-sizing: border-box;
}

/* content-box（デフォルト）: 実際の幅 = 200px + 20px + 2px = 222px */
.content-box {
  box-sizing: content-box;
  width: 200px;
  padding: 10px;
  border: 1px solid;
}

/* border-box: 実際の幅 = 200px（paddingとborderを含む）*/
.border-box {
  box-sizing: border-box;
  width: 200px;
  padding: 10px;
  border: 1px solid;
}

📐 content-box と border-box の違いを図で理解

content-box（デフォルト）width: 200px の場合
content 200px
+
padding 10+10
+
border 1+1
= 実際の幅 222px
border-box width: 200px の場合
content + padding + border
= 実際の幅 200px（宣言通り！）
  • 現代のCSSでは box-sizing: border-box を *（全要素）に適用するのが定番
  • border-boxを使うと「width: 200px」と書いたら画面上でも200pxになるため計算が楽になる

padding・marginのショートハンド

CSS
/* 上下左右すべて同じ */
padding: 16px;

/* 上下 / 左右 */
padding: 16px 24px;

/* 上 / 左右 / 下 */
padding: 8px 16px 24px;

/* 上 / 右 / 下 / 左（時計回り）*/
padding: 8px 16px 24px 32px;

/* auto: ブロック要素を中央揃え */
margin: 0 auto;

🕐 ショートハンドの値の数と意味

padding: 16px → 値が1つ → 上下左右すべて 16px
padding: 16px 24px → 値が2つ → 上下 16px / 左右 24px
padding: 8px 16px 24px → 値が3つ → 上 8px / 左右 16px / 下 24px
padding: 8px 16px 24px 32px → 値が4つ → 時計回り：上・右・下・左
  • 覚え方：TRouBLe（トラブル） → Top・Right・Bottom・Leftの順（時計回り）
  • margin: 0 auto は「上下0・左右auto」。autoは「残りのスペースを均等に」という意味で、横方向の中央揃えに使う

displayプロパティ

CSS
/* block: 前後に改行、幅いっぱいに広がる */
div, p, h1 { display: block; }

/* inline: テキストの流れの中に配置、幅・高さ指定不可 */
span, a, strong { display: inline; }

/* inline-block: インラインで配置、幅・高さ指定可能 */
button { display: inline-block; }

/* none: 要素を非表示（スペースも消える）*/
.hidden { display: none; }

/* flex: Flexboxレイアウト（次のレッスンで詳しく）*/
.container { display: flex; }

/* grid: CSS Gridレイアウト */
.grid { display: grid; }

overflowプロパティ

CSS
.box {
  width: 200px;
  height: 100px;
}

/* visible: はみ出して表示（デフォルト）*/
overflow: visible;

/* hidden: はみ出した部分を非表示 */
overflow: hidden;

/* scroll: スクロールバーを常に表示 */
overflow: scroll;

/* auto: 必要な時のみスクロールバーを表示 */
overflow: auto;

/* テキストを1行で省略（ellipsis）*/
.truncate {
  white-space: nowrap;
  overflow: hidden;
  text-overflow: ellipsis;
}
