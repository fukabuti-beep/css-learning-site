CSS Grid完全ガイド

CSS Gridは2次元レイアウトシステムです。grid-template-columns・fr単位・grid-template-areasをマスターして、複雑なページレイアウトを自在に構築しましょう。

CSS Gridとは

CSS Gridは、行と列の2次元でレイアウトを構築できる強力なシステムです。Flexboxが1次元（行または列）なのに対し、Gridは両方向を同時に制御できます。

CSS
.grid {
  display: grid;
  /* 3列: 各200px */
  grid-template-columns: 200px 200px 200px;
  /* repeat()で簡潔に */
  grid-template-columns: repeat(3, 200px);
  /* fr単位: 使用可能スペースの比率 */
  grid-template-columns: 1fr 2fr 1fr;
  /* 隙間 */
  gap: 1rem;
  row-gap: 2rem;
  column-gap: 1rem;
}

🔍 Grid コードの読み方 - fr単位とrepeat()

.grid { display: grid; grid-template-columns: repeat(3, 1fr); gap: 1rem; }
display: grid → Grid レイアウトを有効化。子要素がグリッドセルに配置される
repeat(3, 1fr) → repeat(繰り返す数, 列の幅)。1frの列を3つ作る
1fr → fr = fraction（分数）。利用可能なスペースを分割する単位。1fr 2fr 1frなら 25% : 50% : 25% の比率
  • Flexbox vs Grid：Flexboxは「アイテムが並ぶ」→ 1次元。Gridは「マス目を作る」→ 2次元
  • 1frは「残りのスペースを1等分」。repeat(3, 1fr)で均等3列グリッドが作れる

repeat() と minmax()

CSS
.grid {
  /* 最小280px、最大1fr の列を自動配置 */
  grid-template-columns: repeat(auto-fill, minmax(280px, 1fr));

  /* auto-fit: 余ったスペースをアイテムが広がって埋める */
  grid-template-columns: repeat(auto-fit, minmax(200px, 1fr));
}

🧩 repeat(auto-fill, minmax()) の読み方

grid-template-columns: repeat(auto-fill, minmax(280px, 1fr));
repeat() → 列の定義を繰り返す関数
auto-fill → コンテナ幅に収まるだけ列を自動的に作る
minmax(280px, 1fr) → 列の幅は最小280px・最大1fr（画面幅に合わせて伸縮）
  • このコード1行でレスポンシブなカードグリッドが完成する「魔法の1行」として有名
  • auto-fillは空のセルを作る、auto-fitは空セルをなくしてアイテムを引き伸ばす違いがある

グリッドアイテムの配置

CSS
.item {
  /* 列の開始〜終了ライン */
  grid-column: 1 / 3;  /* 1〜3ライン（2列分）*/
  grid-column: 1 / -1; /* 全幅 */
  grid-column: span 2; /* 2列分広がる */

  /* 行の開始〜終了ライン */
  grid-row: 1 / 3;
  grid-row: span 2;
}

📐 grid-column: 1 / 3 の読み方

grid-column: 1 / 3 → 列ライン1番から3番まで（＝2列分を占有）
grid-column: 1 / -1 → -1は「最後のライン」。全列幅を占有する
grid-column: span 2 → 現在の位置から2列分広がる（開始ラインは自動）
  • グリッドラインは列の間に引かれる線。3列グリッドは「1・2・3・4」の4本のライン
  • spanを使うと「何列分広げるか」を指定できて直感的

grid-template-areas（名前付きグリッドエリア）

エリアに名前をつけてレイアウトをテキストで表現できます。直感的なレイアウト設計が可能です。

CSS
.layout {
  display: grid;
  grid-template-areas:
    "header header header"
    "sidebar main main"
    "footer footer footer";
  grid-template-columns: 240px 1fr 1fr;
  grid-template-rows: 64px 1fr 60px;
  min-height: 100vh;
}

header  { grid-area: header; }
aside   { grid-area: sidebar; }
main    { grid-area: main; }
footer  { grid-area: footer; }

🗺️ grid-template-areas の読み方

"header header header" → 1行目：3列すべてがheader（横幅いっぱい）
"sidebar main main" → 2行目：1列目がsidebar、2〜3列目がmain
grid-area: header → その要素を「header」という名前のエリアに配置する
  • 文字列でレイアウトを「絵」のように書ける。チームでの意図共有がしやすい
  • コンテナ側でエリア名を定義し、各要素に grid-area: 名前 で割り当てる2段階の仕組み
