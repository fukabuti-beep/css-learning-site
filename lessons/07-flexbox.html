Flexbox完全ガイド

FlexboxはCSSの1次元レイアウトシステムです。justify-content・align-items・flex-growなどのプロパティをマスターして、自在にレイアウトを組みましょう。

Flexboxとは

Flexboxは、1次元（行または列）のレイアウトを効率よく実現するCSSの仕組みです。コンテナに display: flex を指定するだけで、子要素（アイテム）を柔軟に配置できます。

CSS
.container {
  display: flex;           /* Flexboxを有効化 */
  flex-direction: row;   /* row（横）/ column（縦）*/
  gap: 1rem;              /* アイテム間の隙間 */
}

🔍 Flexboxのコード - 1行ずつ読む

.container { display: flex; flex-direction: row; gap: 1rem; }
display: flex → この要素をFlexコンテナにする。子要素が横並びになる
flex-direction: row → 並ぶ方向を指定。row=横、column=縦
gap: 1rem → アイテム同士の隙間を1rem（約16px）にする
  • コンテナとアイテムの区別：display: flexを書く側が「コンテナ（親）」、その直接の子要素が「アイテム（子）」
  • コンテナのプロパティ（justify-content等）とアイテムのプロパティ（flex-grow等）は書く場所が違うので注意

主軸の配置: justify-content

CSS
.container {
  justify-content: flex-start;    /* 左寄せ（デフォルト）*/
  justify-content: flex-end;      /* 右寄せ */
  justify-content: center;        /* 中央寄せ */
  justify-content: space-between; /* 両端揃え */
  justify-content: space-around;  /* 均等配置 */
  justify-content: space-evenly;  /* 完全均等配置 */
}

↔️ 主軸 vs 交差軸 - justify と align の違い

justify-content → 主軸方向（flex-direction と同じ方向）の配置。rowなら横方向
align-items → 交差軸方向（主軸の垂直方向）の配置。rowなら縦方向
  • 覚え方：justify=横に整列、align=縦に整列（flex-direction: rowの場合）
  • space-betweenは「間に均等スペース」、space-aroundは「各要素の周囲に均等スペース」（端の余白が半分）

交差軸の配置: align-items

CSS
.container {
  align-items: stretch;     /* 引き伸ばし（デフォルト）*/
  align-items: flex-start; /* 上揃え */
  align-items: flex-end;   /* 下揃え */
  align-items: center;     /* 垂直中央揃え */
  align-items: baseline;   /* ベースライン揃え */
}

flex-grow / flex-shrink / flex-basis

CSS
.item {
  /* 余ったスペースをどれだけ受け取るか（比率）*/
  flex-grow: 1;

  /* スペース不足時にどれだけ縮むか */
  flex-shrink: 1;

  /* 基本サイズ */
  flex-basis: 200px;

  /* ショートハンド: flex-grow flex-shrink flex-basis */
  flex: 1 1 200px;
  flex: 1;        /* flex: 1 1 0% */
  flex: auto;     /* flex: 1 1 auto */
  flex: none;     /* flex: 0 0 auto（固定サイズ）*/
}

📏 flex ショートハンドの読み方

flex: 1 1 200px; ↑ ↑ ↑ grow shrink basis
flex-grow: 1 → 余ったスペースを「1の比率」で受け取る。0なら広がらない
flex-shrink: 1 → 幅が足りない時に「1の比率」で縮む。0なら縮まない
flex-basis: 200px → 伸び縮みする前の基本サイズ（widthと似た役割）
  • flex: 1 は flex: 1 1 0% の短縮形。複数アイテムに使うと「均等幅」になる
  • flex: none はサイズ固定。ロゴやアイコンなど縮まれると困る要素に使う

実践レイアウト例

① ナビゲーションバー

CSS
.navbar {
  display: flex;
  justify-content: space-between;
  align-items: center;
  padding: 1rem 2rem;
}
.nav-links {
  display: flex;
  gap: 1.5rem;
}

② 完全センタリング

CSS
.center {
  display: flex;
  justify-content: center;
  align-items: center;
  min-height: 100vh;
}
