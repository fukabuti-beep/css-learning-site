CSS変数（カスタムプロパティ）

CSS変数（カスタムプロパティ）を使ってデザイントークンを管理し、ダークモードやテーマ切り替えを実装する方法を学びます。

CSS変数（カスタムプロパティ）とは

CSS変数はCSSの中で値を再利用できる仕組みです。--変数名で定義し、var(--変数名)で参照します。デザイントークンの管理に最適です。

CSS
/* :root でグローバル変数を定義 */
:root {
  --color-primary: #6C63FF;
  --color-secondary: #FF6584;
  --font-size-base: 16px;
  --spacing-md: 1rem;
  --radius: 8px;
}

/* var() で参照 */
.btn {
  background: var(--color-primary);
  padding: var(--spacing-md);
  border-radius: var(--radius);
}

/* フォールバック値（変数が未定義の場合）*/
color: var(--color-accent, #43D9AD);

スコープとカスケード

CSS変数はカスケードと継承に従います。より具体的なセレクターで定義すると、その要素のスコープ内でのみ有効になります。

CSS
:root {
  --color: blue; /* グローバル */
}

.card {
  --color: red;  /* .card内のみ赤 */
  color: var(--color); /* 赤 */
}

.card.highlight {
  --color: gold; /* さらに上書き */
}

ダークモードの実装

CSS
/* ライトモード（デフォルト）*/
:root {
  --bg: #ffffff;
  --text: #1a1a1a;
  --card-bg: #f8f9fa;
}

/* ダークモード: OS設定に従う */
@media (prefers-color-scheme: dark) {
  :root {
    --bg: #0f0e17;
    --text: #fffffe;
    --card-bg: #1e2035;
  }
}

/* ダークモード: クラスで手動切り替え */
[data-theme="dark"] {
  --bg: #0f0e17;
  --text: #fffffe;
}

body {
  background: var(--bg);
  color: var(--text);
  transition: background 0.3s, color 0.3s;
}

JavaScriptからの操作

JavaScript
// CSS変数を取得
const root = document.documentElement;
const color = getComputedStyle(root)
  .getPropertyValue('--color-primary').trim();

// CSS変数を変更
root.style.setProperty('--color-primary', '#FF6584');

// テーマ切り替え
function toggleTheme() {
  document.documentElement.setAttribute(
    'data-theme',
    document.documentElement.dataset.theme === 'dark' ? 'light' : 'dark'
  );
}

calc()との組み合わせ

CSS
:root {
  --spacing: 8px;
  --nav-height: 64px;
}

.main {
  /* ナビバーの高さを除いた残り */
  min-height: calc(100vh - var(--nav-height));
  padding: calc(var(--spacing) * 3); /* 24px */
}
