疑似クラス・疑似要素

疑似クラス（:hover, :nth-childなど）と疑似要素（::before, ::after）を使いこなすことで、HTMLを変更せずにスタイルを高度に制御できます。

疑似クラスとは

疑似クラスは要素の状態に基づいてスタイルを適用します。:hoverや:focusはユーザーの操作状態、:nth-childは位置に基づきます。

CSS
/* ユーザー操作系 */
a:hover    { color: hotpink; }      /* マウスオーバー */
a:visited  { color: purple; }       /* 訪問済み */
input:focus { outline: 2px solid blue; }  /* フォーカス */
button:active { transform: scale(0.97); } /* クリック中 */

/* フォーム状態系 */
input:checked   { }  /* チェックボックスON */
input:disabled  { opacity: 0.5; }
input:invalid   { border-color: red; }
input:required  { }

/* 位置系 */
li:first-child   { }  /* 最初の要素 */
li:last-child    { }  /* 最後の要素 */
li:nth-child(3)  { }  /* 3番目 */
li:nth-child(odd)  { }  /* 奇数番目 */
li:nth-child(2n+1) { } /* 奇数番目（同上）*/
li:nth-child(3n)  { }  /* 3の倍数 */

/* 否定・関係系 */
:not(.disabled)  { }  /* .disabledでない */
:is(h1, h2, h3)  { }  /* どれかに一致 */
:has(img)        { }  /* imgを含む親 */

疑似要素とは

疑似要素は要素の特定の部分、または要素の前後に仮想的な要素を作ります。::（ダブルコロン）を使います。

CSS
/* ::before / ::after — 仮想要素を追加 */
.label::before {
  content: "★ ";  /* content必須！ */
  color: gold;
}

.btn::after {
  content: " →";
}

/* ::placeholder — input/textareaのプレースホルダー */
input::placeholder {
  color: #999;
  font-style: italic;
}

/* ::selection — テキスト選択時 */
::selection {
  background: #6C63FF;
  color: white;
}

/* ::first-line / ::first-letter */
p::first-letter {
  font-size: 3em;
  float: left;
  line-height: 1;
}

::before / ::afterの活用例

CSS
/* アンダーラインアニメーション */
.nav-link {
  position: relative;
  text-decoration: none;
}

.nav-link::after {
  content: "";  /* 空でもOK */
  position: absolute;
  bottom: -2px;
  left: 0;
  width: 0;  /* 初期: 幅0 */
  height: 2px;
  background: var(--primary);
  transition: width 0.3s ease;
}

.nav-link:hover::after {
  width: 100%;  /* ホバーで全幅に */
}

/* カスタムチェックボックス */
.checkbox input:checked + label::before {
  content: "✓";
  background: var(--primary);
  color: white;
}
