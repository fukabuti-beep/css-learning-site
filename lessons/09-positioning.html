ポジショニング

CSSのposition プロパティを使って、要素を自在に配置します。static・relative・absolute・fixed・stickyとz-indexの使い方をマスターしましょう。

positionプロパティ

positionプロパティは、要素をどのように配置するかを制御します。top・right・bottom・leftで位置を指定します。

値説明基準
static通常の流れに従う（デフォルト）—
relative通常位置から相対的にずらす自分の元位置
absolute通常の流れから外れ、指定位置に配置直近のpositioned祖先
fixedビューポートに固定ビューポート
stickyスクロールで追従するスクロールコンテナ

relative と absolute

CSS
/* 親: relativeを設定（absoluteの基準になる）*/
.parent {
  position: relative;
  width: 300px;
  height: 200px;
}

/* 子: 親の右上に配置 */
.badge {
  position: absolute;
  top: -8px;
  right: -8px;
}

fixed と sticky

CSS
/* fixed: ビューポートに固定（ナビゲーションバーなど）*/
.navbar {
  position: fixed;
  top: 0;
  left: 0;
  right: 0;
  z-index: 1000;
}

/* sticky: スクロールで固定される（テーブルヘッダーなど）*/
.sticky-header {
  position: sticky;
  top: 64px;   /* ナビバーの高さ分 */
  z-index: 100;
  background: white;
}

z-index（重なり順）

z-indexはpositionがstatic以外の要素に適用されます。数値が大きいほど手前に表示されます。

CSS
/* z-indexの管理 */
:root {
  --z-base: 0;
  --z-dropdown: 10;
  --z-sidebar: 100;
  --z-nav: 1000;
  --z-modal: 2000;
  --z-toast: 3000;
}

.modal {
  position: fixed;
  inset: 0;  /* top:0; right:0; bottom:0; left:0 */
  z-index: var(--z-modal);
}
