HTMLの基礎

HTMLはWebページの構造を定義するマークアップ言語です。CSSでスタイルを適用する前に、まずHTMLの基本を理解しましょう。

HTMLとは何か

HTML（HyperText Markup Language）は、Webページの構造を作るためのマークアップ言語です。プログラミング言語ではなく、コンテンツの意味と構造を定義するための言語です。1989年にティム・バーナーズ=リーによって考案され、現在も進化し続けています。

ブラウザはHTMLを解釈して、見出し・段落・画像・リンクなどを画面に表示します。CSSで見た目を、JavaScriptで動きを追加することで、本格的なWebサイトが完成します。

💡

HTMLの役割

HTML = 構造（骨格）、CSS = デザイン（見た目）、JavaScript = 動作（機能）

HTMLの基本構造

すべてのHTMLドキュメントは以下の基本構造を持ちます。

HTML
<!-- ドキュメントタイプ宣言 -->
<!DOCTYPE html>

<html lang="ja">
<head>
  <!-- ブラウザへの指示（画面には表示されない） -->
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>ページのタイトル</title>
  <link rel="stylesheet" href="style.css?v=2.0.1">
</head>
<body>
  <!-- ここにページの内容を書く -->
  <h1>こんにちは、World!</h1>
  <p>これは段落です。</p>
</body>
</html>
タグ役割
<!DOCTYPE html>HTML5文書であることをブラウザに伝える
<html lang="ja">HTMLドキュメントのルート要素。言語を指定
<head>メタ情報（タイトル・CSS・文字コードなど）を格納
<body>ページに表示される全コンテンツを格納

主要なHTMLタグ

見出しタグ（h1〜h6）

見出しは重要度順に h1 から h6 まであります。1ページに h1 は1つが基本です。

HTML
<h1>ページの大見出し（最重要）</h1>
<h2>セクション見出し</h2>
<h3>小見出し</h3>
<h4>さらに小さい見出し</h4>

テキスト関連タグ

HTML
<p>段落テキスト</p>
<strong>重要なテキスト（太字）</strong>
<em>強調テキスト（斜体）</em>
<br>  <!-- 改行 -->
<span>インライン要素（スタイル適用に便利）</span>
<div>ブロック要素（グループ化に使用）</div>

リンクと画像

HTML
<!-- リンク -->
<a href="https://example.com" target="_blank">外部リンク</a>
<a href="about.html">内部リンク</a>
<a href="#section">ページ内ジャンプ</a>

<!-- 画像 -->
<img src="photo.jpg" alt="写真の説明" width="400">

リスト

HTML
<!-- 順序なしリスト（ul）-->
<ul>
  <li>リンゴ</li>
  <li>バナナ</li>
  <li>オレンジ</li>
</ul>

<!-- 順序ありリスト（ol）-->
<ol>
  <li>HTMLを学ぶ</li>
  <li>CSSを学ぶ</li>
  <li>JavaScriptを学ぶ</li>
</ol>

セマンティックHTML

セマンティック（意味的）なHTMLとは、コンテンツの意味をタグで表現することです。検索エンジンや支援技術（スクリーンリーダー）が内容を正しく理解できるようになります。

HTML
<header>
  <!-- サイトのヘッダー（ロゴ・ナビなど）-->
  <nav>ナビゲーション</nav>
</header>

<main>
  <!-- ページのメインコンテンツ -->
  <article>
    <!-- 独立したコンテンツ（ブログ記事など）-->
    <section>セクション</section>
  </article>
  <aside>サイドバー・補足情報</aside>
</main>

<footer>フッター（著作権情報など）</footer>

🎯 セマンティックタグ vs divタグ

❌ 非セマンティック

<div id="header">
 <div id="nav">...</div>
</div>
<div id="main">...</div>
<div id="footer">...</div>

✓ セマンティック

<header>
 <nav>...</nav>
</header>
<main>...</main>
<footer>...</footer>

HTML属性

属性（アトリビュート）はタグに追加情報を与えます。属性名="値" の形式で記述します。

HTML
<!-- id: ページ内で一意の識別子 -->
<div id="header"></div>

<!-- class: 複数要素に共通のスタイルを適用 -->
<div class="card active"></div>

<!-- style: インラインCSS（基本的には避ける）-->
<p style="color: red;">赤いテキスト</p>

<!-- data-*: カスタムデータ属性 -->
<button data-id="42" data-action="delete">削除</button>
