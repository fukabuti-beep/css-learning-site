CSSとは何か

CSSはHTMLに見た目を与える言語です。CSSの基本的な書き方と、カスケード・継承・詳細度の概念を学びましょう。

CSSの役割

CSS（Cascading Style Sheets）は、HTMLで構造化されたコンテンツに見た目（スタイル）を与える言語です。色・フォント・サイズ・レイアウトなど、すべての視覚的な要素をCSSで制御します。

1994年にHåkon Wium Lieによって提案され、HTMLとCSSを分離することで保守性が大幅に向上しました。現在はCSS3が普及し、アニメーション・グリッド・カスタムプロパティなど強力な機能を備えています。

CSSの書き方3種類

① 外部スタイルシート（推奨）

別ファイル（.css）にスタイルを書き、HTMLからlinkタグで読み込む方法。最も一般的で推奨される方法です。

HTML
<!-- headタグ内に記述 -->
<link rel="stylesheet" href="css/style.css">

② 内部スタイル（styleタグ）

HTMLファイルの<head>内に<style>タグを使って記述。小規模ページや一時的なスタイルに使用します。

HTML
<head>
  <style>
    h1 {
      color: navy;
      font-size: 2rem;
    }
  </style>
</head>

③ インラインスタイル（非推奨）

HTMLタグに直接style属性で記述。特定要素の一時的なスタイルに使いますが、保守性が下がるため基本的には避けましょう。

HTML
<p style="color: red; font-weight: bold;">赤い太字テキスト</p>
種類場所推奨度用途
外部CSS別ファイル(.css)◎ 推奨本番サイト全般
内部styleheadタグ内△ 可シングルページ・テスト
インラインタグのstyle属性× 非推奨最終手段のみ

CSSの構文

CSSの基本構造は「セレクター」「プロパティ」「値」の3つで構成されます。

🔬 コードの解剖図 — CSSの1ルールを分解する
h1 { color: #6C63FF; /* 文字色 */ font-size: 2.5rem; /* フォントサイズ */ text-align: center; /* テキスト揃え */ }
セレクター スタイルを適用する対象の要素を指定。h1、.card、#header など
プロパティ 何を変えるかを指定するキーワード。color・font-size・margin など
値 どのように変えるかを指定。#6C63FF・2.5rem・center など
{ } と ; 波括弧{ }でルールを囲み、セミコロン;で宣言を区切る

コードを読む際は次の順序で確認しましょう：

コード
意味・役割
h1
ページ内すべての <h1> タグに適用する（セレクター）
color
文字の色を変えるプロパティ
#6C63FF
紫色の16進数カラーコード（値）
font-size: 2.5rem
文字サイズを基準フォントの2.5倍に設定（1rem = 通常16px）
text-align: center
テキストを中央揃えにする
CSS
/* セレクター { プロパティ: 値; } */

h1 {
  color: #6C63FF;      /* 文字色 */
  font-size: 2.5rem;   /* フォントサイズ */
  text-align: center;  /* テキスト揃え */
  margin-bottom: 1rem; /* 下の余白 */
}

.card {
  background: white;
  border-radius: 8px;
  padding: 1.5rem;
  box-shadow: 0 2px 8px rgba(0,0,0,0.1);
}
💡 2つ目のルール .card を読み解く
1
.card — ドット（.）から始まるのはクラスセレクター。HTML側で class="card" と書いた要素に適用される
2
background: white; — 背景色を白に設定。white はCSSの色キーワード（#ffffff と同じ）
3
border-radius: 8px; — 角を8pxの半径で丸める。値を大きくするほど角が丸くなり、50% で円になる
4
padding: 1.5rem; — 要素の内側の余白を上下左右に1.5rem（約24px）設定する
5
box-shadow: 0 2px 8px rgba(0,0,0,0.1); — 影を追加。X方向 Y方向 ぼかし 色 の順。rgba(0,0,0,0.1)は10%不透明の黒

カスケード・継承・詳細度

カスケード（Cascade）

「Cascading」は「流れ落ちる」という意味。複数のCSSルールが同じ要素に適用される場合、どのルールが優先されるかを決める仕組みです。

継承（Inheritance）

親要素のスタイルが子要素に引き継がれる仕組みです。colorやfont-sizeは継承されますが、marginやborderは継承されません。

CSS
body {
  font-family: 'Noto Sans JP', sans-serif;
  color: #333;
  /* body内のすべての要素に継承される */
}

p {
  /* 明示的に指定しない限り body の color を継承 */
  font-size: 1rem;
}

.special {
  color: inherit;   /* 明示的に親の値を継承 */
  color: initial;  /* ブラウザの初期値にリセット */
}
💡

開発者ツールを活用しよう

ブラウザのF12（開発者ツール）→ Elements → Styles タブで、どのCSSルールが適用されているか確認できます。打ち消し線が引かれているルールは上書きされています。
