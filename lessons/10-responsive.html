レスポンシブデザイン

メディアクエリとモバイルファーストの考え方で、あらゆるデバイスに対応するレスポンシブデザインを実装しましょう。

レスポンシブデザインとは

レスポンシブデザインとは、スマートフォン・タブレット・PCなど様々な画面サイズに対して、一つのHTMLで最適な表示を提供するデザイン手法です。2010年にEthan Marcotteが提唱しました。

HTML
<!-- 必須: viewportメタタグ -->
<meta name="viewport"
      content="width=device-width, initial-scale=1.0">

メディアクエリ

メディアクエリは「画面幅がXpx以上/以下の時に特定のスタイルを適用する」ための構文です。

CSS
/* モバイルファースト: min-widthで大きい画面に向けて追加 */
.container {
  padding: 1rem;     /* モバイル（デフォルト）*/
}

@media (min-width: 768px) {
  .container {
    padding: 2rem;   /* タブレット以上 */
  }
}

@media (min-width: 1024px) {
  .container {
    padding: 3rem;   /* デスクトップ */
  }
}

/* 他のメディア特性 */
@media (orientation: landscape) { }   /* 横向き */
@media (prefers-color-scheme: dark) { }  /* ダークモード */
@media print { }                        /* 印刷時 */

相対単位とclamp()

CSS
/* clamp(最小値, 推奨値, 最大値) */
h1 {
  font-size: clamp(1.5rem, 5vw, 3rem);
  /* 最小1.5rem、画面幅の5%、最大3rem */
}

.container {
  width: min(100%, 1200px);
  margin: 0 auto;
  padding: 0 max(1rem, 5vw);
}

/* レスポンシブなグリッド（メディアクエリ不要）*/
.grid {
  display: grid;
  grid-template-columns: repeat(auto-fit, minmax(280px, 1fr));
  gap: 1.5rem;
}

モバイルファーストアプローチ

モバイルファーストとは、最初にモバイル向けのスタイルを書き、min-widthで大きな画面向けのスタイルを追加していく考え方です。

💡

なぜモバイルファースト？

スマートフォンからのアクセスが過半数を占める現代では、モバイルを優先した設計が重要です。また、小さい画面から設計することで、コンテンツの優先度を明確にできます。
