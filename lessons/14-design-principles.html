デザイン原則

CRAPデザイン原則・視覚的ヒエラルキー・カラー理論・アクセシビリティ・余白設計など、プロのWebデザイナーが実践するデザイン原則を学びます。

CRAPデザイン原則

Robin Williamが提唱したCRAPは、4つのデザイン原則の頭文字です。この原則を理解することで、見やすく美しいデザインが作れます。

🔗

C — 近接（Contrast）
※近接=Proximity

関連する要素をグループ化し、無関係な要素は離す。ページの構造を明確にする。

📐

R — 整列（Repetition）
※整列=Alignment

すべての要素をグリッドや見えない線に沿って配置。ランダムな配置を避ける。

🔄

A — 反復（Alignment）
※反復=Repetition

デザイン要素（色、フォント、形）を繰り返して一貫性を生む。スタイルガイドの基本。

⚡

P — コントラスト（Proximity）
※コントラスト=Contrast

異なる要素を明確に区別する。色・サイズ・フォント・形で差をつける。

視覚的ヒエラルキー

視覚的ヒエラルキーとは、情報の重要度を視覚的に伝える技術です。ユーザーの目が自然に重要な情報へ誘導されます。

CSS
/* タイポグラフィスケール（モジュラースケール）*/
:root {
  --ratio: 1.25;     /* Major Third */
  --base: 1rem;

  --text-sm:   0.8rem;    /* 12.8px */
  --text-base: 1rem;      /* 16px */
  --text-lg:   1.25rem;   /* 20px */
  --text-xl:   1.5625rem; /* 25px */
  --text-2xl:  1.953rem;  /* 31px */
  --text-3xl:  2.441rem;  /* 39px */
  --text-4xl:  3.052rem;  /* 49px */
}

60-30-10カラールール

プロのデザイナーが使う配色の基本ルールです。

60% — ベースカラー

背景、大きなエリア。落ち着いた色。

30% — サブカラー

サイドバー、カード。ベースと同系色。

10% — アクセントカラー

ボタン、リンク、強調。目立つ色。

アクセシビリティ（WCAG）

基準コントラスト比対象
AA（最低基準）4.5:1以上通常テキスト
AA（大きなテキスト）3:1以上18pt/14pt Bold以上
AAA（推奨）7:1以上通常テキスト
CSS
/* アクセシビリティ対応のfocus */
:focus-visible {
  outline: 3px solid #6C63FF;
  outline-offset: 3px;
  border-radius: 4px;
}

/* スクリーンリーダー専用テキスト */
.visually-hidden {
  position: absolute;
  width: 1px;
  height: 1px;
  padding: 0;
  margin: -1px;
  overflow: hidden;
  clip: rect(0, 0, 0, 0);
  white-space: nowrap;
  border: 0;
}

ホワイトスペース（余白）の重要性

💡

余白は「デザインの呼吸」

余白は何もない空間ではなく、レイアウトの重要な構成要素です。要素間の余白を統一することで、整然とした美しいデザインが生まれます。余白が少なすぎると窮屈に、多すぎると間が抜けた印象になります。

CSS
/* スペーシングシステム（8pxベース）*/
:root {
  --space-1:  0.25rem;  /*  4px */
  --space-2:  0.5rem;   /*  8px */
  --space-3:  0.75rem;  /* 12px */
  --space-4:  1rem;     /* 16px */
  --space-6:  1.5rem;   /* 24px */
  --space-8:  2rem;     /* 32px */
  --space-12: 3rem;     /* 48px */
  --space-16: 4rem;     /* 64px */
}
