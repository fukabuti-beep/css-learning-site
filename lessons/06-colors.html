色と背景

色の指定方法（HEX・RGB・HSL）、背景プロパティ、グラデーション、filterまでWebデザインに欠かせない色の知識を学びます。

色の指定方法

CSS
/* キーワード */
color: red;
color: transparent;

/* HEX（16進数）*/
color: #6C63FF;     /* フル */
color: #6C63FF80;   /* アルファ付き（50%透明）*/
color: #fff;        /* 短縮形 */

/* RGB */
color: rgb(108, 99, 255);
color: rgba(108, 99, 255, 0.5); /* アルファ付き */

/* HSL（色相・彩度・明度）*/
color: hsl(245, 100%, 69%);
color: hsla(245, 100%, 69%, 0.5);

/* oklch（モダンブラウザ対応・知覚的に均一）*/
color: oklch(0.65 0.2 260);

背景プロパティ

CSS
.hero {
  /* 背景色 */
  background-color: #0f0e17;

  /* 背景画像 */
  background-image: url('hero.jpg');

  /* 画像サイズ */
  background-size: cover;   /* contain / 100% / auto */

  /* 画像位置 */
  background-position: center;

  /* 繰り返し */
  background-repeat: no-repeat;

  /* スクロール挙動 */
  background-attachment: fixed; /* パラレックス効果 */
}

グラデーション

CSS
/* linear-gradient（直線グラデーション）*/
background: linear-gradient(135deg, #6C63FF, #43D9AD);
background: linear-gradient(to right, #667eea, #764ba2);

/* radial-gradient（放射状グラデーション）*/
background: radial-gradient(circle at center, #6C63FF, transparent);

/* conic-gradient（扇形グラデーション）*/
background: conic-gradient(from 0deg, #6C63FF, #43D9AD, #6C63FF);

/* 複数の背景を重ねる */
background:
  linear-gradient(rgba(0,0,0,0.5), rgba(0,0,0,0.5)),
  url('photo.jpg') center/cover;

filterプロパティ

CSS
img {
  filter: blur(4px);           /* ぼかし */
  filter: brightness(0.8);     /* 明度 */
  filter: contrast(1.2);       /* コントラスト */
  filter: grayscale(100%);     /* グレースケール */
  filter: saturate(2);         /* 彩度 */
  filter: drop-shadow(2px 2px 8px rgba(0,0,0,0.5));
}

/* backdrop-filter: 背景へのフィルター */
.glass {
  background: rgba(255,255,255,0.1);
  backdrop-filter: blur(20px);
}
