タイポグラフィ

フォントの種類・サイズ単位（px/em/rem）・テキストプロパティを学び、読みやすく美しいタイポグラフィを実現しましょう。

フォントファミリー

CSSでフォントを指定する際は、フォントスタックを使って複数のフォントを優先順に指定します。

CSS
body {
  /* フォントスタック: 左から優先順位 */
  font-family: 'Noto Sans JP', 'Hiragino Kaku Gothic ProN', sans-serif;
}

/* 3種類の汎用フォントファミリー */
.serif    { font-family: Georgia, serif; }       /* 明朝体系 */
.sans     { font-family: Arial, sans-serif; }    /* ゴシック体系 */
.mono     { font-family: 'Courier New', monospace; } /* 等幅フォント */

Google Fontsの使い方

HTML
<!-- 1. headタグ内でフォントを読み込む -->
<link href="https://fonts.googleapis.com/css2?family=Noto+Sans+JP:wght@400;700&display=swap" rel="stylesheet">
CSS
/* 2. CSSで使用する */
body {
  font-family: 'Noto Sans JP', sans-serif;
}

font-sizeの単位

単位説明用途
pxピクセル（絶対単位）ボーダー幅など固定値
em親要素のfont-sizeに対する相対値padding・marginなど
remルート要素(html)のfont-sizeに対する相対値フォントサイズ全般
%親要素に対するパーセンテージ幅・高さの指定
vw/vhビューポートの幅・高さに対する割合全画面レイアウト
CSS
html { font-size: 16px; } /* ベースサイズ */

h1 { font-size: 2.5rem; }  /* 16px × 2.5 = 40px */
h2 { font-size: 2rem; }    /* 16px × 2 = 32px */
p  { font-size: 1rem; }    /* 16px × 1 = 16px */
small { font-size: 0.875rem; } /* 14px */

/* clamp(): 最小値・推奨値・最大値で流動的サイズ */
h1 { font-size: clamp(1.5rem, 5vw, 3rem); }
💡

rem を使う理由

remはhtmlのfont-sizeを基準にするため、ユーザーがブラウザのフォントサイズを変更しても対応できます。アクセシビリティの観点からもremの使用が推奨されます。

テキストプロパティ一覧

CSS
p {
  /* 太さ: 100〜900または名前 */
  font-weight: 700;        /* bold */

  /* スタイル */
  font-style: italic;     /* 斜体 */

  /* 行の高さ（可読性に重要）*/
  line-height: 1.7;        /* font-sizeの1.7倍 */

  /* 文字間隔 */
  letter-spacing: 0.05em; /* 少し広げる */

  /* 単語間隔 */
  word-spacing: 0.1em;

  /* テキスト揃え */
  text-align: left;       /* center / right / justify */

  /* 下線・打ち消し線など */
  text-decoration: none;  /* underline / line-through */

  /* 大文字・小文字変換 */
  text-transform: uppercase; /* lowercase / capitalize */
}

見出しと本文のサイズ比を統一することで、視覚的な一貫性が生まれます。

🎯 スケールのデモ

h2 / 2rem / 32px

h3 / 1.5rem / 24px

h4 / 1.125rem / 18px

本文テキスト / 1rem / 16px — line-height: 1.7 で読みやすく

補足テキスト / 0.875rem / 14px
